stockfish library: https://pypi.org/project/stockfish/ 
How to run this code:
1. Download stockfish from this link: https://stockfishchess.org/download/ 
2. Install python-dotenv: pip install python-dotenv 
3. Create a .env file and define your filePath variable
    - your filepath might look like this filepath = "yourdirectories/stockfish/stockfish-windows-x86-64-avx2.exe"
2. Replace file path with your own path
